# 💳 Cloud Pricing — UI Layout Project

Современная адаптация блока тарифов для SaaS-сервиса, реализованная на чистом **HTML5 и CSS3** без использования фреймворков и библиотек.

Проект демонстрирует навыки верстки, работу с Flexbox, аккуратную архитектуру CSS и внимание к деталям интерфейса.

---

## 📌 О проекте

Данный проект представляет собой секцию с тарифными планами:

* **Personal**
* **Business**
* **Developer (Most Popular)**
* **Agency**

Основная задача — реализовать чистый, визуально сбалансированный и аккуратный UI с градиентами, иерархией контента и интерактивными элементами.

---

## 🎯 Что демонстрирует этот проект

### 🔹 Семантическая HTML-структура

* Четкое разделение секций (`section`, `ul`, `li`)
* Логичная вложенность элементов
* Понятная структура документа

### 🔹 Архитектура CSS

* BEM-подобный нейминг классов
* Использование CSS-переменных (`--card-accent`)
* Разделение глобальных и локальных стилей
* Повторно используемые компоненты

### 🔹 Работа с Flexbox

* Горизонтальное выравнивание карточек
* Контроль внутреннего позиционирования элементов
* Гибкое распределение пространства

### 🔹 UI-детали

* Градиентные фоны для каждого тарифа
* Выделение “Most Popular” через абсолютное позиционирование
* Hover-эффект кнопок с динамической сменой цвета
* Иконки чек-листов через `::before`

## 💡 Инженерные решения

* Цвет акцента каждой карточки управляется через CSS-переменную.
* Кнопки используют `var(--card-accent)` для динамической смены цвета при hover.
* Верстка построена через Flexbox без хаотичного позиционирования.
* Контент структурирован так, чтобы легко масштабировать количество тарифов.
